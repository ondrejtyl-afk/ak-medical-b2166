## Deploy na Netlify

### Metoda 1: Drag & Drop
1. Jděte na [app.netlify.com/drop](https://app.netlify.com/drop)
2. Přetáhněte tuto složku (`deploy/`)
3. Hotovo!

### Metoda 2: ZIP
```bash
cd deploy && zip -r ../akmedical-web.zip .
```
